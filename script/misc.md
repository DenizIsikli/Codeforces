#### Open the terminal (:term) - Optional if you need to use *iostream* in your cpp file
```bash
:term
```

#### Make the script executable
```bash
chmod +x [script_name].sh
```

#### Compile the script with a given file
```bash
./[script_name].sh /path/to//path/to/file.cpp```
```

#### Run the executable
```bash
./[script_name]
```
